== Description ==
TheCartPress Product Options is a plugin for TheCartPress. It allows to add one or two options levels to the products, such as sizes, colors...

= TheCartPress Product Options =

* Allows to edit the options by a admin pages
* Allows to set the custom view of the options: One or Two Selection lists
* Supports Multi Lingual

= Unlimited features =

* TheCartPress-productoptions is compatible with TheCartPress and has been developed by TheCartPress Team.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload TheCartpress Product Options to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click the new link 'options' in the list of products
